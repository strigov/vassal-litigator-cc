# update-index — Обновление индекса

Когда пользователь запускает эту команду:

1. Убедись что `.vassal/index.yaml` существует. Если нет — предложи `/vassal-litigator:init-case` и остановись.
2. Прочитай `skills/update-index/SKILL.md`. Следуй Фазе 1 (Сканирование и Preview, read-only).
3. Покажи результат сканирования Сюзерену. Жди подтверждения действий.
4. После подтверждения: прочитай `skills/codex-invocation/SKILL.md`. Следуй Фазе 3 (Apply via Codex medium --write).
5. После терминации Codex: следуй Фазе 4 (Верификация). Покажи Сюзерену резюме.
