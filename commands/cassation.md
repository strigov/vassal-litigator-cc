# cassation -- Кассационная жалоба

Когда пользователь запускает эту команду с "$ARGUMENTS":

1. Прочитай скилл `skills/cassation/SKILL.md` и следуй его инструкциям.
2. Если `$ARGUMENTS` содержит основания обжалования -- используй как приоритетные.
3. Если `$ARGUMENTS` содержит пути к судебным актам -- загрузи.
4. Убедись, что `.vassal/case.yaml` существует.
5. Загрузи решение, постановление апелляции и полный контекст дела.
6. Делегируй анализ Opus-субагенту (model: "opus").
7. Выполни pipeline: загрузка контекста -> анализ судебных актов -> поиск оснований -> стресс-тест -> preview -> генерация текста -> .docx -> обновление состояния.
