# catalog — Каталогизация документов

Когда пользователь запускает эту команду:

1. Убедись что `.vassal/index.yaml` существует. Если нет — предложи `/vassal-litigator:init-case` и остановись.
2. Прочитай `skills/catalog/SKILL.md`. Следуй Фазе 1 (Preview, read-only): определи документы без `summary`, покажи план.
3. Покажи preview Сюзерену. Жди подтверждения.
4. После подтверждения: прочитай `skills/codex-invocation/SKILL.md`. Следуй Фазе 3 (Apply via Codex medium --write).
5. После терминации Codex: следуй Фазе 4 (Верификация). Покажи резюме и путь к `xlsx`.
