# appeal -- Апелляционная жалоба

Когда пользователь запускает эту команду с "$ARGUMENTS":

1. Прочитай скилл `skills/appeal/SKILL.md` и следуй его инструкциям.
2. Если `$ARGUMENTS` содержит основания обжалования -- используй как приоритетные.
3. Если `$ARGUMENTS` содержит путь к решению суда -- загрузи.
4. Убедись, что `.vassal/case.yaml` существует.
5. Загрузи решение суда и полный контекст дела.
6. Делегируй анализ решения Opus-субагенту (model: "opus").
7. Выполни pipeline: загрузка контекста -> анализ решения -> поиск оснований -> стресс-тест -> preview -> генерация текста -> .docx -> обновление состояния.
