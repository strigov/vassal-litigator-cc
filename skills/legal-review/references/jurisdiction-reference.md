# Справочник: Подсудность арбитражных дел

## 1. Компетенция арбитражных судов (ст. 27-33 АПК РФ)

### Критерии арбитражной подведомственности
- **Субъектный:** обе стороны — юридические лица или ИП
- **Предметный:** экономические споры и иные дела, связанные с предпринимательской деятельностью

### Специальная подведомственность (ст. 33 АПК) — независимо от субъектов:
- Банкротство
- Корпоративные споры
- Споры об отказе в госрегистрации юрлиц/ИП
- Споры о деятельности депозитариев
- Споры по интеллектуальным правам (Суд по интеллектуальным правам)

## 2. Территориальная подсудность

### Общая подсудность (ст. 35 АПК)
Иск подаётся в арбитражный суд по месту нахождения ответчика (юр. адрес из ЕГРЮЛ/ЕГРИП).

### Подсудность по выбору истца (ст. 36 АПК)
| Случай | Суд | Основание |
|--------|-----|-----------|
| Ответчик — за границей или адрес неизвестен | по месту нахождения имущества или последнему известному адресу | ч. 1 ст. 36 |
| Несколько ответчиков в разных местах | по месту нахождения любого из них | ч. 2 ст. 36 |
| Ответчик — в иностранном государстве | по месту нахождения истца или имущества ответчика | ч. 3 ст. 36 |
| Иск из договора с указанием места исполнения | по месту исполнения | ч. 4 ст. 36 |
| Иск из причинения вреда имуществу | по месту причинения вреда | ч. 6 ст. 36 |
| Иск к перевозчику | по месту нахождения перевозчика | ч. 3 ст. 38 |

### Договорная подсудность (ст. 37 АПК)
- Стороны **могут изменить** общую и альтернативную подсудность **соглашением**
- Пророгационная оговорка: «все споры рассматриваются в {суд}»
- Обычно включается в раздел «Разрешение споров» договора
- **Не может изменить** исключительную подсудность и подсудность по субъекту

### Исключительная подсудность (ст. 38 АПК) — НЕ МОЖЕТ быть изменена
| Категория | Суд |
|-----------|-----|
| Иски о правах на недвижимость | по месту нахождения недвижимости |
| Иски о правах на морские/воздушные суда | по месту государственной регистрации |
| Иски к перевозчику из договора перевозки | по месту нахождения перевозчика |
| Корпоративные споры | по месту нахождения юрлица |
| Заявления о признании должника банкротом | по месту нахождения должника |
| Споры об обжаловании решений третейских судов | по месту рассмотрения третейского дела |

## 3. Чек-лист проверки подсудности

### Шаг 1: Компетенция (арбитражный или СОЮ?)
- [ ] Обе стороны — юрлица/ИП
- [ ] Спор экономический / из предпринимательской деятельности
- [ ] Или попадает в специальную подведомственность (ст. 33)

### Шаг 2: Исключительная подсудность
- [ ] Проверить, не подпадает ли спор под ст. 38 АПК (недвижимость, корпоративные, банкротство)
- [ ] Если да — подсудность определена, дальше не проверяем

### Шаг 3: Договорная подсудность
- [ ] Проверить раздел «Разрешение споров» в договоре
- [ ] Есть ли пророгационная оговорка (какой именно суд указан?)
- [ ] Оговорка не противоречит исключительной подсудности

### Шаг 4: Общая / альтернативная подсудность
- [ ] Если нет договорной оговорки → по месту нахождения ответчика
- [ ] Проверить, есть ли основания для альтернативной подсудности (ст. 36)

### Шаг 5: Если дело уже подано
- [ ] Верно ли определена подсудность?
- [ ] Нет ли оснований для передачи в другой суд (ст. 39 АПК)?
- [ ] Не заявил ли оппонент ходатайство о передаче?

## 4. Третейская оговорка

Если в договоре есть третейская (арбитражная) оговорка:
- Проверить, действительна ли она (конкретность, наименование третейского суда/арбитражного учреждения)
- Если третейская оговорка действительна → арбитражный суд оставит иск без рассмотрения (п. 5 ч. 1 ст. 148 АПК)
- **Исключения:** оговорка недействительна, утратила силу, не может быть исполнена
- Для возражения: сторона должна заявить до первого заявления по существу спора

## 5. Частые проблемы

1. **Пророгационная оговорка указывает несуществующий суд** → оговорка недействительна, применяется общая подсудность
2. **Оговорка в пользу суда одной стороны** → действительна (это законно, если не нарушает исключительную подсудность)
3. **ИП утратил статус после заключения договора** → спор может уйти в СОЮ
4. **Множественность ответчиков** → истец выбирает суд по месту нахождения любого
5. **Встречный иск** → подаётся в суд, рассматривающий основной иск
