# Шаблон MD-зеркала

MD-зеркало — это полнотекстовое markdown-представление документа дела. Хранится в `.vassal/mirrors/{id}.md`. Это кэш извлечённого текста для экономии токенов, а не замена юридически значимого оригинала.

## Формат

```markdown
---
id: doc-001
source: "Материалы от клиента/Договоры/2025-06-01 ООО Ромашка Договор поставки №47.pdf"
type: договор
title: "Договор поставки №47"
date: 2025-06-01
pages: 12
extraction_method: pdf-text
extraction_model: haiku
extraction_date: 2026-03-26
confidence: high
---

# Договор поставки №47

## от 01 июня 2025 года

[Полный текст документа...]

---
<!-- EXTRACTION NOTES -->
<!-- Страница 7: таблица извлечена приблизительно -->
```

## Поля frontmatter

| Поле | Обязательное | Описание |
|------|-------------|----------|
| id | да | ID документа (doc-NNN), совпадает с index.yaml |
| source | да | Путь к оригиналу (от корня папки дела) |
| type | да | Тип документа (из таксономии) |
| title | да | Название документа |
| date | нет | Дата документа |
| pages | нет | Количество страниц |
| extraction_method | да | pdf-text, ocr, docx-parse, image-read, manual |
| extraction_model | нет | Модель, выполнившая извлечение (haiku, sonnet) |
| extraction_date | да | Дата создания зеркала |
| confidence | да | high, medium, low |

## Правила создания

1. **Полный текст** — весь текст переносится с сохранением структуры (заголовки, пункты, таблицы)
2. **Таблицы** → markdown-таблицы
3. **Изображения** → `[Изображение: описание]`
4. **Нечитаемые фрагменты** → `[Нечитаемо: описание проблемы]`
5. **HTML-комментарии** — только технические пометки о качестве извлечения
6. **Юридические выводы** — НЕ помещать в зеркало (место для них — `analysis/` и `positions/`)
7. **Размер документа** — не основание для усечения. Зеркало содержит весь извлечённый текст, даже если документ объёмный. Если OCR/парсер извлёк текст частично, в зеркало попадает ровно то, что фактически извлечено, без дополнительного усечения.

## Методы извлечения (extraction_method)

- `pdf-text` — PDF с текстовым слоем, извлечён программно
- `ocr` — распознан через tesseract/ocrmypdf
- `docx-parse` — извлечён из .docx через python-docx или Read tool
- `image-read` — изображение прочитано через LLM (Haiku vision)
- `manual` — текст введён вручную

## Уровни уверенности (confidence)

- `high` — текст извлечён чисто, структура сохранена
- `medium` — есть мелкие артефакты, но текст читаем
- `low` — существенные проблемы, требуется ручная проверка (ставь `needs_manual_review: true` в индексе)
